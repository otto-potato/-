云上柜周 — AI智能衣柜
一句话介绍
一款由AI全链路驱动的智能衣柜App——录入衣物后，AI结合天气自动推荐每日穿搭。
核心功能
- **AI衣物分析**：手动录入衣物基本信息，AI生成详细描述存入衣柜
- **智能穿搭推荐**：每日获取天气数据，AI根据已有衣物推荐最佳搭配
- **穿着状态追踪**：自动记录穿着次数，支持标记清洗状态
- **定时推荐**：每天凌晨3点自动生成穿搭通知
技术栈
Kotlin + Jetpack Compose + Room + Hilt + WorkManager + Retrofit/OkHttp
快速开始
1. 安装APK，允许定位权限
2. 设置 → 配置AI模型（支持DeepSeek/OpenAI兼容API）
3. 添加衣物 → AI分析 → 保存
4. 首页 → AI推荐 → 获取穿搭建议
天气源
默认Open-Meteo（免费免配置），失败自动回退。也支持OpenWeatherMap、WeatherAPI、和风天气。

| 参数 | 值 |
|------|-----|
| API地址 | https://api.deepseek.com |
| 模型 | deepseek-v4-flash |
